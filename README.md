# Hostrail Releases

This public repository is the official binary distribution and update feed for
[Hostrail](https://github.com/Raterfy/hostrail-releases/releases/latest).

Hostrail is a local-first SSH, SFTP, terminal, and port-forwarding workspace for
macOS, Windows, Linux, and Android. It does not require a Hostrail account,
subscription, analytics service, cloud synchronization, or proprietary backend.

## Downloads

Open the [latest release](https://github.com/Raterfy/hostrail-releases/releases/latest)
and download the artifact matching your platform. Verify downloaded files with
the release's `SHA256SUMS.txt`.

The application checks this repository's public GitHub Releases API at most once
every 24 hours when automatic update checks are enabled. No account or tracking
identifier is sent by Hostrail.

This repository contains release metadata and binaries only; it is not the
Hostrail source repository.
